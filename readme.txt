== Description ==

This plugin is for Microsoft/Bing Ads conversion tracking in WooCommerce. It will allow you to add your Microsoft/Bing Ads tracking code to a WooCommerce site in order to track conversion events.

== Installation ==

Install, activate, follow the instructions on the plugin page to enter your Microsoft UET code and you should be set!

== Frequently Asked Questions ==

<strong>Do I need to instal the general tracking code separately?</strong>
No, the plugin includes both the general UET Tag code and the conversion tracking event code, so there is no need to add either of these yourself.

<strong>Does work with different currencies?</strong>
Yes, you can change which currency you would like to use in the plugin's settings.

<strong>Does this work with the latest version of WooCommerce?</strong>
It's been tested and works in both WooCommerce v3 and v4.
